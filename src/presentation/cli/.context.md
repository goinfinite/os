Command-line interface for Infinite OS using Cobra framework. Provides the same functionality as the REST API but through shell commands, useful for scripting and automation.

## Summary

- cli.go — CLI initialization, root command setup
- router.go — Command registration for all features
- controller/ — Command handlers for each domain
- init/ — Database service initialization for CLI
- middleware/ — CLI-specific middleware (environment checks, root privilege enforcement)

## Constraints

- MUST require root/elevated privileges (enforced in middleware)
- MUST validate all inputs before calling use cases
- Commands MUST follow naming conventions established in router.go

## Guidance

- SHOULD provide parallel functionality to REST API
- SHOULD support both long-form (--flag) and short-form (-f) flags
- SHOULD provide helpful error messages for invalid input
