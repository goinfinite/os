CLI-specific middleware for panic handling, root access checking, environment validation, and logging setup.

## Summary

- PanicHandler — Catches and recovers from panics
- PreventRootless — Ensures CLI runs with elevated privileges
- CheckEnvs — Validates required environment variables
- LogHandler — Configures logging for CLI operations

## Constraints

- MUST prevent execution without root privileges
- MUST validate all required environment variables at startup
