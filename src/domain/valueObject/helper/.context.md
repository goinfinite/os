Helper functions for type conversions and interface handling. These utilities enable safe conversion from interface{} (common in unmarshaling and dynamic data) to strongly-typed primitives.

## Summary

- interfaceToBool.go — Convert interface{} to bool with error handling
- interfaceToInt64.go — Convert interface{} to int64
- interfaceToString.go — Convert interface{} to string
- interfaceToUint.go, interfaceToUint16.go, interfaceToUint32.go, interfaceToUint64.go — Unsigned integer conversions
- findNamedGroupsMatches.go — Regex named group extraction
- strongStringHasher.go — Secure string hashing

## Constraints

- MUST return errors for invalid conversions rather than panicking
- MUST NOT lose type information during conversion

## Guidance

- SHOULD use these helpers when unmarshaling API payloads or configuration data
