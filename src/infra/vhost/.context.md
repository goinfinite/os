Virtual host configuration and management. Manages NGINX vhost configurations for hosting multiple applications and mappings. Owns all nginx-specific operations including config validation, reload, and primary virtual host resolution.

## Summary

- Virtual host configuration generation
- NGINX configuration management
- Vhost lifecycle (create, update, delete)
- NGINX config validation and reload
- Primary virtual host hostname resolution (env, nginx conf, system fallback)
- VirtualHostHelpers for nginx operations (config validation, reload, hostname resolution, server_name updates)

## Constraints

- MUST generate valid NGINX configuration
- MUST reload NGINX after configuration changes
- MUST NOT depend on infra/runtime (orchestration belongs in domain use cases)
